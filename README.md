# 📚 Library Management System

A simple Python-based Library Management System that helps manage books, members, and library transactions efficiently. The project automates common library operations such as searching, issuing, and returning books while maintaining accurate records.

---

## 📖 Introduction

The **Library Management System (LMS)** is designed to simplify library operations by managing books, users, and transactions in an organized manner. It reduces paperwork, minimizes human errors, and improves the overall efficiency of library management.

---

## 📝 Description

The system allows librarians to:

- Store and manage book information
- Maintain member records
- Search books by title or author
- Issue books to members
- Return books
- Track available and borrowed books
- Calculate fines for late returns

## ⚙️ How It Works

1. Admin adds books and member details.
2. Users search for books using the title or author name.
3. Books are issued to members.
4. Records are updated automatically.
5. Returned books become available again.
6. Fine is calculated automatically for late returns.

---

## ▶️ How to Run

1. Install Python 3 on your computer.
2. Download or clone this repository.
3. Open the project folder.
4. Run the Python file using:

```bash
python library_management.py
```
